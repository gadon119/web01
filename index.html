WEB

A web page (also written as webpage) is a document that acts as a web resource on the World Wide Web. In order to display a web page, a web browser is needed to retrieve web pages from the Internet. When accessed by a web browser it may be displayed as a web page on a monitor or mobile device. Typical web pages are hypertext documents which contain hyperlinks, often referred to as [1]links, for browsing to other web pages.
